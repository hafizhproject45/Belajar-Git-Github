Cara git :
-Pwd ( untuk melihat directory sekarang )
-Cd ( untunk merubah directory )
-git init ( untuk menjadi lokal repo )
-cd .. ( keluar directory )
-mkdir <nama_file> ( untuk membuat file baru )

Cara pertama :
-git add  .
-git commit -m ""

Jika sudah modified :
-git commit -am ""

Liat graph branch :
-git log  --all --decorate --oneline --graph
( graph )

Pindah branch :
-git checkout <nama_branch>

Chenk branch :
-git branch

Cara liat yang dapat di delete :
-git branch --merged

Deletenya :
-git branch -d <nama_branch>

Three way merge :
-git merge <nama-branch>

Pindah branch ke yg sudah di hapus :
-git checkout <hash_7>

Git Remote :
-git remote -v ( untuk mengecek isi dari remote )
-git push ( untuk menambahkan file yang belum ter simpan di github )

Melihat Config :
-git config --list ( untuk melihat config yang sudah di buat )
-git config --global <user.name "" dan user.email ""> ( mengubah untuk semua repo )
-git config --local <user.name "" dan user.email ""> ( mengubah untuk repo yang sedang digunakan saja )

Konflik di remote dan local yang lebih duluan :
-git fetch ( untuk melihat yang lebih dulu mana)
-git pull ( untuk menarik file yang tidak sama yang ada di remote )

Jika sudah selesai koreksi konflik :
-git add .
-git commit -m ""
-git push
